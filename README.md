# Angular ViewChild component interaction

This example demonstrates how to use Angular's ViewChild decorator to access and interact with child components and DOM elements from a parent component. It shows how to get a reference to a child component instance and call its methods, simulating interaction with DOM elements.

## Language

`html`

## How to Run

Save the code as an HTML file (e.g., index.html) and open it in a web browser. No build process or external dependencies are required.
